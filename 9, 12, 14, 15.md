# Задание 9
### Какие компании организуют перелеты из Владивостока (Vladivostok)?
### Поля в результирующей таблице: name
## Решение:
```
SELECT DISTINCT Company.name
from Company
JOIN Trip ON Company.id = Trip.company
WHERE Trip.town_from="Vladivostok"
```
## Вывод:
|   | name      |
|---|-----------|
| 1 | Dale_avia |
